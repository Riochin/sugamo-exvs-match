# Product Overview

身内12名専用の「機動戦士ガンダムEXVS 2 Infinite Boost」月例下剋上決定戦管理Webアプリ。ゲームセンターでの対戦終了直後（10〜15分）に全員のスマートフォンを使って利用する。

## Core Capabilities

1. **同期型結果発表**: 全員の成績入力完了を契機に、全スマホが一斉に結果発表画面へ強制遷移する。
2. **焦らし演出**: 順位を「1軍残留」→「2軍残留」→「ボーダー（入れ替え対象）」の順で段階的に開示し、eスポーツ大会のような緊張感を演出する。
3. **Star投票**: 1大会につき各プレイヤーが3つのStarを任意の対象（複数分配 or 全ツッパ）に贈る称賛システム。自己投票は禁止。
4. **簡易ログイン**: プレイヤー名選択 + 4桁PIN入力による認証。厳密なセキュリティよりも筐体前での手軽さを優先。
5. **グループ管理**: 1軍（1st TEAM）・2軍（2nd TEAM）の所属状況と個人プロフィール（称号・主使用機体・直近5回の勝率推移）の閲覧。

## Target Use Cases

- **プライマリ**: 大会終了直後、全員がその場でスマホを取り出して結果発表を同時に視聴する「公開処刑」セレモニー。
- **セカンダリ**: 過去の大会履歴や個人プロフィールの閲覧（日常的な確認用途）。

## Value Proposition

単なる成績集計ツールではなく、「eスポーツ大会のような熱狂体験」を身内グループにもたらすエンタメ装置。勝敗を超えてStar投票という称賛の仕組みを持つことで、全員がゲームとして楽しめる場を提供する。
